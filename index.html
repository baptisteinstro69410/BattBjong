Scan BJONG – recherche SN + date

Application HTML mobile pour iPhone. Le scan est limité au format Code 128. La permission caméra iOS est demandée avant le chargement de ZXing.

Code 128 iPhone / Safari Recherche sur A ou B
Prêt. Appuyez sur « Autoriser & scanner ».

Astuce iPhone : ouvrez cette page dans Safari et hébergez-la en HTTPS (ou localhost). L’ouverture directe depuis l’app Fichiers peut bloquer la caméra.

La recherche essaie d’abord la valeur exacte, puis la variante avec / sans préfixe V.
